## Этап: идея сайта

Создаётся статический сайт про жёлтую Toyota MR-S 2000 года.

Машина условно называется «Желтик».

Цель сайта — представить автомобиль, его историю, фотографии и информацию в удобном виде.

## Этап: блок qr-sign

Обсуждался HTML-блок qr-sign.

Для проверки остатков блока использовалась команда:

    Select-String -Path "src\template.html","index.html","en\index.html","zh\index.html" -Pattern "qr-sign|qrTitle|qrText|undefined"

После удаления блока из src/template.html нужно обязательно запускать сборку:

    node src/build.js

---

## Этап: ошибка с удалением блока

Был важный негативный опыт: ассистент предложил слишком широкий способ удаления HTML-блока, что могло удалить лишний код.

Итоговое правило:

- не использовать опасные regex-команды удаления без проверки;
- сначала смотреть найденный фрагмент;
- затем удалять вручную или очень точечно.
