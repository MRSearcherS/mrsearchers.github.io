# Открытые задачи проекта MRS

Дата обновления: 2026-07-11 23:37:09

## Назначение файла

Этот файл хранит текущие незакрытые задачи.

Его нужно обновлять после завершения задач.

---

## 1. Проверить актуальное состояние сайта

Проверить:

- открывается ли сайт по адресу https://mrs.spb.ru/;
- работает ли HTTPS;
- нет ли предупреждений GitHub Pages;
- корректно ли открываются языковые версии.

Проверить адреса:

- https://mrs.spb.ru/
- https://mrs.spb.ru/en/
- https://mrs.spb.ru/zh/

---

## 2. Проверить GitHub Pages

Проверить в настройках репозитория:

- выбранную ветку публикации;
- кастомный домен;
- статус DNS;
- статус HTTPS;
- наличие галочки Enforce HTTPS, если доступна.

---

## 3. Проверить файл CNAME

В репозитории должен быть файл CNAME.

Ожидаемое содержимое:

    mrs.spb.ru

---

## 4. Проверить сборку

После изменений выполнить:

    node src/build.js

Затем проверить, что обновились:

- index.html
- en/index.html
- zh/index.html

---

## 5. Проверить остатки qr-sign

Проверить, нет ли в шаблоне и собранных файлах остатков проблемного блока или undefined-переменных:

    Select-String -Path "src\template.html","index.html","en\index.html","zh\index.html" -Pattern "qr-sign|qrTitle|qrText|undefined"

---

## 6. Поддерживать CURRENT_CODE.md в актуальном состоянии

После значимых изменений обновлять файл:

docs/assistant-context/CURRENT_CODE.md

Он должен содержать актуальные версии:

- src/build.js
- src/template.html
- src/locales/ru.json
- src/locales/en.json
- src/locales/zh.json
- CNAME

---

## 7. Проверить изображения

Проверить:

- реальные имена файлов изображений;
- отсутствие двойных расширений;
- корректность путей в HTML/CSS/JSON;
- отображение изображений на GitHub Pages.

---

## 8. Подготовить финальный QR-код

Если домен и HTTPS работают стабильно, подготовить QR-код на:

https://mrs.spb.ru/

Для печати предпочтительно SVG.

---

## 9. Перед серьёзными Git-действиями проверять статус

Перед push/rebase/reset/merge проверять:

    git status

Если есть сомнения — сначала показать вывод ассистенту.
